ClickyChrome
A Chrome Extension for Clicky Web Analytics

Use

You must enter the following information below for each site you want to monitor:

  1. Name (Required): Name for your site, used in drop down menus. Example: 'My Blog'.
  2. Domain (Optional): Site domain as registered with Clicky. This is used to enable context menu integration. No need to enter the 'https://', and no trailing slashes. Example: 'test.com' or 'www.test.com'.
  3. Site ID (Required): The Site ID can be found in the site preferences area on Clicky.
  4. Site Key (Required): The Site Key can be found in the site preferences area on Clicky.
