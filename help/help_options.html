Clicky Chrome Options

Badge Color
Not much to explain, what color do you like?
Badge Count
What do you want the counter on the ClickyChrome icon to keep track of? You can have it display the number of visitors currently online, the total number of visitors today, or the total number of goals completed today.
Goal Notification

You can be notified via HTML5 desktop notifications whenever a visitor completes a goal.

View example notification.

When a goal is completed, it will display the following information:

  • List of goals the visitor has completed in this session
  • Total value of goals completed
  • Visitor's geolocation, as well as IP or custom username you have assigned
  • Start time of visit

When more than one visitor has completed a goal, you'll see an arrow in the title bar that you can click to scroll through the visitors.

Notifications can be set to display for either 10, 20 or 30 seconds before disappearing. However, if you hover over a notification it will stay visible until you manually close it.

The example notification above will not automatically expire, you have to manually close it.

Goal checking and badge counters update every minute when actively using your browser.
