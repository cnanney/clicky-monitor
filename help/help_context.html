Context Menu Integration

If you enter a site domain, ClickyChrome will add an item to your context menu (when you right-click on a page) that will allow you to "View page stats." Just as the name implies, you will be taken to the Clicky stats page for whatever page of your site you were viewing.

For example, if I'm browsing my website and am viewing the page example.com/some-cool-post.html, I can right-click on the page, select "View page stats," and a new tab will open with the stats page for some-cool-post.html.

If you want to remove the context menu for a site, simply delete the domain field for that site.
